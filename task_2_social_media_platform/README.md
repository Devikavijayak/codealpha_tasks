# Mini Social Media Platform

A mini social media platform built for a fast and premium internship assignment. 

## Features
- **User Profiles:** View profile stats and follow other users.
- **Posts & Comments:** Create posts and add comments to existing ones.
- **Like/Follow System:** Like posts and follow users. The feed shows your posts and posts from users you follow.
- **Rich Aesthetics:** Modern dark mode design with neon accents and skeleton loading animations.

## Tech Stack
- **Frontend:** Vanilla HTML, CSS, JavaScript (No external frameworks like React/Vue for maximum speed and simplicity).
- **Backend:** Node.js with Express.js
- **Database:** SQLite3

## How to Run Locally

1. **Install Dependencies:**
   Make sure you have Node.js installed. Run the following command in the terminal:
   ```bash
   npm install
   ```

2. **Start the Server:**
   ```bash
   node server.js
   ```

3. **Open the App:**
   Open your browser and navigate to `http://localhost:3000`.
